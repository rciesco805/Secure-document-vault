# Fund Offering Documents

This folder contains the official fund offering documents for investor review.

## Required Documents

Place the following PDF files in this directory:

1. **LPA.pdf** - Limited Partnership Agreement
   - The legal agreement between the General Partner and Limited Partners
   - Outlines rights, responsibilities, and profit/loss allocation

2. **PPM.pdf** - Private Placement Memorandum
   - Disclosure document containing investment risks, terms, and fund strategy
   - Required for 506(c) compliance

3. **Subscription-Agreement.pdf** - Subscription Agreement
   - The contract investors sign to commit capital to the fund
   - Contains investor representations and commitments

## Instructions

1. Obtain finalized versions of each document from legal counsel
2. Rename files exactly as shown above (case-sensitive)
3. Place PDF files in this folder: `/public/fund-documents/`
4. Documents will be accessible at:
   - https://dataroom.bermudafranchisegroup.com/fund-documents/LPA.pdf
   - https://dataroom.bermudafranchisegroup.com/fund-documents/PPM.pdf
   - https://dataroom.bermudafranchisegroup.com/fund-documents/Subscription-Agreement.pdf

## Security Note

These documents are publicly accessible via direct URL. For sensitive documents requiring authentication, use the secure dataroom upload feature instead.
